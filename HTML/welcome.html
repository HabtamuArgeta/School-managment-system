Here is the basic gaide how to use the system

In order to to add data

  1. click on Add button at the top center of the page
  2. Select your an Object (teacher, student, course, registration, adminstration, school, or class)
  3. fill all fields which is required
  4. Click on Insert

In order to to Update data

  1. click on update button at the top center of the page
  2. Select your an Object (teacher, student, course, registration, adminstration, school, or class)
  3. Search object by object id
  4. Modify the field as you wish
  5. click on Update
